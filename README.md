# webfollow-app

> RSS reader suppot fever api

- Strive to be the best reader experience
- 致力成为体验最佳的阅读器

## Feature

- [x] text-reader
- [x] podcast-reader
- [x] img-reader
- [x] video-reader
- [x] column-view
- [x] list-view
- [x] card-view
- [x] fever api
- [x] data in local
- [x] recommend
- [x] llm summary
- [x] pwa
- [ ] export opml
- [ ] data reset
